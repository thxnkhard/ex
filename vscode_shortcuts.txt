[Basic editing 기본 편집]

Ctrl + X
 : Cut line (empty selection)
 잘라내기

Ctrl + C
 : Copy line (empty selection)
 복사하기

Alt + ↓/↑
 : Move line down/up
 줄 옮기기 (다중 선택 후 옮기기도 가능)

Shift + Alt + ↓/↑
 : Copy line down/up
 줄 복사하기 (다중 선택 후 복사도 가능)

Shift + Ctrl + K
 : Delete line
 줄 삭제

Ctrl + Enter / Shift + Ctrl + Enter
 : Insert line below/above
 줄 삽입 후 이동(아래/위)

Shift + Ctrl + \
 : Jump to matching bracket
 연결된 괄호로 이동


Ctrl + ] / Ctrl + [
: Indent/outdent line
 들여쓰기/내어쓰기

Home / End
 : Go to beginning/end of line
 줄의 처음으로/끝으로

Ctrl + ↑ / Ctrl + ↓
 : Go to beginning/end of file
 해당 파일 스크롤

Ctrl + PgUp / Ctrl + PgDn
 : 이전/다음 파일 보기

Alt + PgUp / Alt + PgDn
 : Scroll page up/down
 페이지 스크롤 올리기 / 내리기

PgUp / PgDn
 : 커서를 페이지 제일/아래로 위로 옮김 (스크롤 가능)

Alt + Ctrl [ / Alt + Ctrl ]
 : Fold/unfold region

Ctrl + K Ctrl [ / Ctrl + K Ctrl ]
 : Fold/unfold all subregions

Ctrl + K Ctrl + 0 / Ctrl + K Ctrl + J
 : Fold/unfold all regions
 모든 region 접기/펴기

Ctrl + K Ctrl + C
 : Add line comment

Ctrl + K Ctrl + U
 : Remove line comment

Ctrl + /
 : Toggle line comment

Shift + Alt + A
 : Toggle block comment

Alt + Z
 : Toggle word wrap
 자동 줄 바꿈
 
F11
 : Toggle full screen





[Multi-cursor and selection 다중 커서 및 다중 선택]

Alt + Click
 : Insert cursor
 다중 커서

Alt + Ctrl + ↑/↓
 : Insert cursor above/below
 커서 위/아래에 넣기

Ctrl + U
 : Undo last cursor operation
 이전 커서를 작동시킴

Shift + Alt + I
 : Insert cursor at end of each line selected
 각각 선택된(드래그한) 줄(들)의 마지막에 커서 넣기

Ctrl + L
 : Select current line
 현재 줄 선택

Ctrl + Shift + L
 : Select all occurrences of Find match
 일치하는 모든 항목 선택

Ctrl + F2
 : Change all occurrences
 모든 항목 변경

Shift + Ctrl + ← / →
 : Expand / shrink selection
 커서 다음에 나오는 단어 선택

Ctrl + ← / →
 : 커서 다음에 나오는 단어로 이동

Shift + ← / →
 : 커서 다음에 나오는 한 글자 선택

Shift + Alt + drag mouse
 : Column (box) selection
 다중 행 선택

Shift + Alt + Ctrl + ↑ / ↓
 : Column (box) selection up/down
 다중 행 선택 (위/아래) 한줄씩

Shift + Alt + Ctrl + ← / →
 : Column (box) selection left/right
 다중 행 선택 (좌/우) 한글자씩

Shift + Alt + Ctrl + PgUp
 : Column (box) selection page up
 다중 행 선택 (페이지 업)

Shift + Alt + Ctrl + PgDn
 : Column (box) selection page down
 다중 행 선택 (페이지 다운)
